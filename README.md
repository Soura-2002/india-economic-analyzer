# India Economic Trends Analyzer

Interactive dashboard showing India GDP, Inflation, and Unemployment (1991-2023).

Built with **React 19 + TypeScript + Recharts + Tailwind CSS v4 + Vite 7**.

## Features
- GDP, Inflation & Unemployment trends (1991-2023)
- Interactive area, bar, and comparison charts
- Year-range filtering with dual sliders
- Auto-generated insights
- 2-year linear regression forecast
- Downloadable PDF report

## Run locally
```bash
pnpm install
pnpm --filter @workspace/india-economic-analyzer run dev
```

Data source: World Bank / IMF estimates
